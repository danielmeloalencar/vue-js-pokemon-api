### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/danielmeloalencar/vue-js-pokemon-api.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm run serve
   ```
